## Markdown City Oita 
Everything in the world should be described in Markdown.

## Markdown

検索エンジンがチャトボット UI になるとしたら、web サイトは訪問されることを期待するのではなく、シンプルなテキストでの情報提供によるメッセージの伝達を期待すべきです。（2023 年 7 月時点）

  • フォーマット
    • ヒューマンフレンドリー
    • AI フレンドリー
      • ChatGPT も Markdown を理解している
  • 軽量でシンプル
    • プレーンなテキストファイル
    • 誰でも開くことができる
    • 共有しやすい
  • 充実したツール
    • web アプリやモバイルアプリが多数
    • Notion や Slack も Markdown を採用
    • Mermaid 記法で図を埋め込むことができるツールも
  • 様々な形式に変換
    • HTML, PDF, Word 形式などに変換可能
    • HTML に変換して web に公開
